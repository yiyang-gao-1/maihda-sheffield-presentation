Three Innovative Applications Using Understanding Society Data

Research Question

"How do physical health trajectories differ across intersectional groups defined by sex, ethnicity, and education (fixed characteristics), and how do these trajectories vary with aging and time?"

Data & Variables

Dataset

Understanding Society
Waves 1-14 (2009-2023)

Sample Size

62,155
individuals

Observations

465,115
total measurements

Outcome

SF-12 PCS
Physical Component Score (0-100)

Key Variables

Variable Type Description
stratum_fixed Fixed Grouping Sex × Ethnicity × Education (40 groups)
sf12pcs_dv Outcome Physical health score (higher = better)
age_centered Time-varying Age centered at 50 years
wave_centered Time-varying Survey wave (centered at wave 1)
birth_cohort Fixed Birth cohort (Pre-1940 to 1990s+)

Model Progression

Model 0: Empty Model

sf12pcs_dv ~ 1 + (1|pidp) + (1|stratum_fixed)
Baseline ICC calculation: Individual ICC = 61.79%, Stratum ICC = 7.03%

Model 1: Basic Time Effects

sf12pcs_dv ~ wave_centered + age_centered + (1|pidp) + (1|stratum_fixed)
Added: wave + age effects
Age effect = -0.242 points/year, Wave effect = -0.033 points/wave

Model 2: Age Trajectories

sf12pcs_dv ~ wave_centered * age_centered + (1|pidp) + (1|stratum_fixed)
Added: wave × age interaction
Health decline accelerates with age (interaction = -0.008, p<0.001)

Model 3: Fixed Effects

sf12pcs_dv ~ wave_centered + age_centered + sex_fixed + eth_fixed + edu_fixed + (1|pidp) + (1|stratum_fixed)
Added: sex + ethnicity + education
98.6% reduction in stratum variance (PCV)

Model 4: Random Slopes

sf12pcs_dv ~ wave_centered + age_centered + sex_fixed + eth_fixed + edu_fixed + (1 + wave_centered|pidp) + (1|stratum_fixed)
Added: Individual-specific trajectories
Negative correlation (-0.387) between baseline health and decline rate

Model 5: Full Model

sf12pcs_dv ~ wave_centered + age_centered + sex_fixed + eth_fixed + edu_fixed + birth_cohort + wave_centered:age_centered + (1 + wave_centered|pidp) + (1|stratum_fixed)
Added: Birth cohort effects and interactions
Final stratum ICC = 0.11% (near zero)

Key Findings

0.11%
Final stratum variance (from 7.03%)
3.4 points
Education gradient (Degree vs None)

Key Insight

Intersectional effects are minimal after accounting for additive effects. Health inequalities operate through additive disadvantage rather than unique emergent properties of specific sex-ethnicity-education combinations. Individual heterogeneity dominates over group-level differences.

Research Question

"To what extent do geographic contexts amplify or mitigate mental health inequalities across intersectional groups?"

Data & Variables

Dataset

Wave 14
Cross-sectional analysis

Sample Size

33,977
individuals

Geographic Units

25
Region × Urban/Rural

Intersectional Strata

246
groups

Key Variables

Variable Type Description
sf12mcs_dv Outcome Mental health score (0-100, higher = better)
stratum Grouping Sex × Ethnicity × Education × Age
spatial_unit Grouping Region × Urban/Rural (e.g., "R7_Urban" = London Urban)
area_deprivation Area-level Standardized composite deprivation score

Model Progression

Model 1: Cross-Classified Null Model

sf12mcs_dv ~ 1 + (1|stratum) + (1|spatial_unit)
Stratum ICC = 8.2%, Spatial ICC = 0.5%

Model 2: Individual-Level Effects

sf12mcs_dv ~ sex_cat + eth_cat + edu_cat + age_cat + (1|stratum) + (1|spatial_unit)
Added: sex + ethnicity + education + age
Stratum PCV = 75.4%

Model 3: Area-Level Effects

sf12mcs_dv ~ sex_cat + eth_cat + edu_cat + age_cat + area_deprivation + (1|stratum) + (1|spatial_unit)
Added: area_deprivation
Each unit increase in deprivation → 0.536 point decrease in MCS

Model 4: Cross-Level Interactions

sf12mcs_dv ~ sex_cat + eth_cat + edu_cat + age_cat + area_deprivation + edu_cat:area_deprivation + (1|stratum) + (1|spatial_unit)
Tested: education × area_deprivation
No significant interaction (p = 0.78)

Key Findings

1.8 points
Urban-rural mental health gap
2.4 points
Regional variation (best to worst)
0.5%
Spatial variance (very limited)
6 areas
Priority areas identified

Key Insight

Geographic effects are minimal compared to individual and intersectional effects. Rural areas show 1.8 points better mental health than urban areas. Only 0.5% of variance is attributable to geographic areas, suggesting that place-based interventions alone may have limited impact.

Research Question

"How did COVID-19 differentially impact mental health across intersectional groups, and which groups were most vulnerable to mental health deterioration?"

Baseline

Pre-COVID (2019-2020)

COVID Waves

9 surveys
April 2020 - March 2021

Sample Size

Observations

111,101

Key Variables

Variable Type Description
scghq1_dv Outcome GHQ-12 score (0-36, higher = worse mental health)
ghq_change Outcome COVID GHQ - Baseline GHQ
deteriorated Binary Outcome Clinical deterioration (change ≥ 4 points)
lockdown Context "Lockdown 1", "Lockdown 2", "No lockdown"

Model Progression

Model 1: Null Model for Change Score

ghq_change ~ 1 + (1|stratum) + (1|pidp)
Individual ICC = 66.07%, Stratum ICC = 1.02%

Model 2: Individual Characteristics

ghq_change ~ sex_cat + eth_cat + edu_cat + age_cat + (1|stratum) + (1|pidp)
Added: demographics predicting change
Stratum PCV = 91%

Model 3: Baseline Mental Health Effects

ghq_change ~ sex_cat + eth_cat + edu_cat + age_cat + baseline_ghq + baseline_income + (1|stratum) + (1|pidp)
Added: baseline_ghq + baseline_income
Baseline GHQ coefficient = -0.439 (regression to mean)

Model 4: Interaction Model

ghq_change ~ sex_cat + eth_cat + edu_cat + age_cat + baseline_ghq + edu_cat:baseline_ghq + baseline_income + (1|stratum) + (1|pidp)
Tested: differential vulnerability
Significant interaction - impact varies by education and baseline health

Model 5: Longitudinal Recovery Model

ghq_score ~ baseline_ghq + time + sex_cat + eth_cat + edu_cat + age_cat + (1 + time|pidp) + (1|stratum)
Growth curve for recovery
Minimal overall recovery trend (-0.003 points/month)

Key Insight

COVID-19 mental health impacts were highly heterogeneous across intersectional groups. Recovery patterns varied significantly, with some groups showing persistent effects.
